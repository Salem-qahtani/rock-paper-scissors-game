# Rock Paper Scissors

A browser-based Rock Paper Scissors game built with HTML, CSS, and JavaScript.

## Live Preview

[View on GitHub Pages](https://salem-qahtani.github.io/rock-paper-scissors-game/)

## About

This project is an interactive Rock Paper Scissors game where the player competes against the computer. Click an icon to make your choice, and the game tracks scores for both sides. Built as part of [The Odin Project](https://www.theodinproject.com/) Foundations course.

## Features

- **Choice Icons** — Clickable rock, paper, and scissors buttons with Font Awesome icons
- **Score Tracking** — Live player and computer score display
- **Result Display** — Color-coded win/lose/draw feedback
- **Reset Button** — Restart the game at any time

## What I Learned

- DOM manipulation with `querySelectorAll`, `getElementById`, and `addEventListener`
- Using `Math.random()` and `Math.floor()` to generate random computer choices
- Game logic with conditional statements
- Dynamically updating text content and styles based on game results
- CSS transitions for hover and active states
