# Rock Paper Scissors

A browser-based Rock Paper Scissors game built with HTML, CSS, and JavaScript.

## Live Preview

[View on GitHub Pages](https://salem-qahtani.github.io/rock-paper-scissors-game/)

## About

This project is an interactive Rock Paper Scissors game where the player competes against the computer. Click an icon to make your choice, and the game tracks scores for both sides. Built as part of The Odin Project Foundations course.

## Features

- **Choice Icons** — Clickable rock, paper, and scissors buttons
- **Score Tracking** — Live player and computer score display
- **Result Display** — Color-coded win/lose/draw feedback
- **Reset Button** — Restart the game at any time

## Built With

- HTML
- CSS (Flexbox)
- JavaScript (DOM Manipulation)
- [Font Awesome](https://fontawesome.com/)
